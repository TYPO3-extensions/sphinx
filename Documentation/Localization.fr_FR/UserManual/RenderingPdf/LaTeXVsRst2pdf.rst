Comparaison LaTeX / rst2pdf
^^^^^^^^^^^^^^^^^^^^^^^^^^^

Dans un souci de vous convaincre que le rendu LaTeX est réellement beaucoup plus élégant et professionnel, nous avons
généré la documentation que vous êtes en train de lire une fois avec ``rst2pdf`` :

|pdf_rst2pdf|

et ensuite avec LaTeX :

|

|pdf_latex|

Les fichiers ont été générés sans configuration ou réglages spéciaux.

.. note::
	Un inconvénient de LaTeX est qu'il a besoin d'être installé globalement sur votre système ; c'est la raison pour
	laquelle cette extension ne propose pas un assistant dans le gestionnaire d'extensions pour installer et configurer
	LaTeX automatiquement pour vous.
